Claude 提示框架九步驟

The Claude Prompt Framework

九個編號步驟，依序帶你寫出精準、可交付的提示——先定核心公式，再餵參考檔案、對齊語氣，最後在執行前確認規則與共識。

  • 核心公式：〔動作動詞〕＋〔明確交付物〕＋〔以便收件人在期限前完成 X〕，並標明範圍、長度、格式。
  • 先讓 Claude 完整讀參考檔案、比照其結構與語氣的「節奏」，再開始回應。
  • 明確指定輸出長度、收件人預期反應、要動用的工具（搜尋／Gmail／Drive／Calendar）。
  • 執行前先請它列出最關鍵的 3 條規則與 5 步計畫；不清楚就提問，不要默默補空白。

原作：Redgate — The Claude Prompt Framework。中文重製版，依原圖風格重繪。
